# New to dbt?

Lightdash connects to your existing dbt project to manage all your data in one place.

With dbt you can transform the data in your data warehouse so it's ready to explore in Lightdash.

If you haven't used dbt before, [follow dbt's getting started guide](https://docs.getdbt.com/tutorial/setting-up) 
before proceeding with installing Lightdash.
