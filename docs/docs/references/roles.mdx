# Roles and permissions

:::caution

This document is a draft of an upcoming feature

:::

* Everybody in your organization will join as an `Organization Member` by default unless specified.
* All organization members can create their own projects and will be the Project Admin for that project.

### Project Roles

| Action                                | Project Admin | Project Editor | Project Viewer  |
|:--------------------------------------|:-------------:|:--------------:|:---------------:|
| View charts and dashboards            |       ✅       |       ✅        |        ✅        |
| Run metric queries                    |       ✅       |       ✅        |        ✅        |
| Create/edit charts and dashboards     |       ✅       |       ✅        |        ❌        |    
| Create/edit metrics                   |       ✅       |       ✅        |        ❌        |
| Run SQL queries                       |       ✅       |       ✅        |        ❌        |
| Manage project access and permissions |       ✅       |       ❌        |        ❌        |
| Delete project                        |       ✅       |       ❌        |        ❌        |

### Organization Roles


| Action                                     | Organization Admin | Organization Editor | Organization Viewer | Organization Member |
|:-------------------------------------------|:------------------:|:-------------------:|:-------------------:|:-------------------:|
| Invite users to organization               |         ✅          |          ✅          |          ✅          |          ✅          |
| Create new projects                        |         ✅          |          ✅          |          ✅          |          ✅          |   
| View **all** projects                      |         ✅          |          ✅          |          ✅          |          ❌          |
| Edit **all** projects                      |         ✅          |          ✅          |          ❌          |          ❌          |
| Admin for **all** projects                 |         ✅          |          ❌          |          ❌          |          ❌          |
| Manage organization access and permissions |         ✅          |          ❌          |          ❌          |          ❌          |

## Organization members

Everybody has these permissions by default. They can invite new `Organization Members` and create their own projects.

| Action                                  | Scope |
|-----------------------------------------|-------|
| Invite users to organization            | Organization |
| Create Projects | Organization |

## Project Viewers

Project viewers have the ability to view all content in a project and play with the explorer but cannot edit or
create any new content.

| Action                      | Scope |
|-----------------------------|---|
| View charts / dashboards    | Project |
| Execute queries in explorer | Project |

## Project Editors

Project editors have full access to running queries and creating content within a project but cannot change any
project settings.

| Action                        | Scope |
|-------------------------------|---|
| View project (see previous)   | Project |
| Create charts / dashboards    | Project |
| Edit metrics / dimensions     | Project |
| Execute queries in SQL Runner | Project |

## Project Admins

Project admins have full control over projects, their settings, and access to other members of the Organization.

| Action                        | Scope |
|-------------------------------|---|
| Edit project (see previous)   | Project |
| Delete project                | Project |
| Manage access and permissions | Project |

## Organization Viewer

Organization viewers have view access to all projects and content in the organization.

| Action                                 | Scope |
|----------------------------------------|----|
| Organization member (see above)        | Organization |
| View all projects (see Project Viewer) | Organization |

## Organization editor

Organization Editors have editor access to all projects and content in the organization.

| Action                                 | Scope |
|----------------------------------------|----|
| Organization viewer (see previous)     | Organization |
| Edit all projects (see Project Editor) | Organization |

## Organization admin

Organization admins have full admin access to all projects, organization access controls, and destructive actions.

| Action                                  | Scope |
|-----------------------------------------|----|
| Organization Editor (see previous)      | Organization |
| Manage all projects (see Project Admin) | Organization |
| Manage access and permissions | Organization |
| Manage organization settings | Organization |
